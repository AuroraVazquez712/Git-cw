Etiquetas HTML -mi referencia
Etiqueta Para qué sirve Ejemplo
title Título de la página en el navegador 
 <title>Paginita web 2</title>
body Cuerpo de la página: todo lo que se va a ver 
 
<body> 
    <p>Esta es mi página</p> 
</body>
h1 Encabezado 1 que da jerarquía 
 <Título de nivel h1>
p Insterar párrafo 
 <párrafo empieza en una línea nueva>
pre Muestra el texto tal cual se pone en el html 
<pre><code> 
#include < stdio.h>
int main ()
{
    printf("Hola CW!");
    return 0:
}
</code></pre>
ul Lista desordenada o por viñetas 
<ul>
    <li>Elemento 1</li>
    <li> Elemento 2</li>
    <li>Elemento 3</li>
</ul>
ol Lista ordenada con alfabeto 
<ol style="list-style-type: lower-alpha;">>
        <li>Elemento 1</li>
        <li> Elemento 2</li>
        <li>Elemento 3</li>
</ol>
