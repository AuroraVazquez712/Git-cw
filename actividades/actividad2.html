Explicación linea a línea

Pero vamos paso por paso 


    int main()

Esta línea inicia la función principal 


    int edad ()

Declaramos la variable edad con un entero 


    printf("Cuántos años tienes?");

Mostramos en pantalla la frase que verá el usuario para ingresar su edad 


    scanf ("%d", &edad);

Guardamos el valor ingresado por el usuario en la variable edad 


    if (edad >= 18)

Condicional para saber si el valor ingresado por el ususario y guardado previamente en la variable edad, es mayor o igual a 18 


     printf("Eres mayor de edad, \n");

Si la edad fue mayor a 18 mostrará en pantalla la siguiente línea


    else

En caso de que el condicional no se cumpla, entonces...


    printf("Eres menor de edad \n");

Se despliega en pantalla esta línea


    return 0;

Fin de la función regresará una acción

¿Qué hace el código en general?

El código le pide al ususario inrgesar su edad, si esta es mayor a 18 le despliega un mensaje de mayoría de edad, y en caso de ser menor de minoría de edad
